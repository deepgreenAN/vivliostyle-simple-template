# vivliostyleドキュメントのテンプレート

themeは`vivliostyle.config.js`内で変更する．作成時は`docs/index.md`に記述する．

## requirements

- vivliostyle-cli
- sass
- (nu)

```sh
just dev
```

- プレビュー

```sh
just preview
```

- ビルド

```sh
just build
```

- デモ

```sh
just demo
```
