# vivliostyleドキュメントのテンプレート

themeは`vivliostyle.config.js`内で変更する．作成時は`docs/index.md`に記述する．

## requirements

- vivliostyle-cli(npmでインストール)
- sass
- nu
- pueue
- just

## コマンド

- 開発モード

```sh
just dev
```

- ビルド

```sh
just build
```

- デモ

プレビュー

```sh
just preview_demo
```

ビルド

```sh
just build_demo
```
